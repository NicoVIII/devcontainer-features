# Devcontainer Features

[![Last commit](https://img.shields.io/github/last-commit/NicoVIII/devcontainer-features?style=flat-square)](https://github.com/NicoVIII/devcontainer-features/commits/)

This repository contains a collection of features to use inside of devcontainers written in Amber compiled to bash.
To have a look at the available features, check out the [src](./src) directory. Each feature contains its own README.md with instructions on how to use it.
